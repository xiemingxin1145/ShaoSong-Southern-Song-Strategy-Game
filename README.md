# 绍宋：南渡无悔 - 南宋深度策略游戏项目

**GitHub Repo for Southern Song Dynasty Strategy Game Design & Prototype**

参考：《无悔入华夏》、《三国志》系列 + 《绍宋》小说架空历史元素。

由 Grok 自主完成一条龙搜索整理、设计文档与起步原型。

## 项目概览
- **类型** : 策略 SLG / Grand Strategy Lite (回合制大地图 + 官员管理 + 城市经营 + 政治派系)
- **背景** : 南宋 (1127-1279)，从建炎南渡到崖山海战。可选架空模式（参考《绍宋》）或纯历史。
- **特色** : 千名武将文臣数据库（核心历史 + 生成扩展），百张精美插图（AI批量生成管道），城池地图、河流航道、海军系统。
- **目标** : 统一江南、抗金/抗蒙、改革朝政、经济开发。

## 快速开始
1. 查看 `GDD.md` 完整游戏设计文档（包含机械、系统、历史资料、艺术方向）
2. `data/` 文件夹有城市、官员样例JSON、事件列表
3. `prototype/` 简单HTML5可玩原型（地图 + 官员列表 + 基本模拟）
4. `art/` 插图资源与生成提示

由主人授权后自主完成，后续可扩展为完整游戏项目。
