# 《绍宋：南渡无悔》 多城市实时渲染地图与多国家扩展系统 v1.0

**主人授权后自主添加** ：多城市实时渲染动态地图 + 更多参考国家/势力系统。

## 一、 多城市实时渲染地图核心设计

** 核心目标 ** : 玩家能实时感知南宋多城市状态、部队移动、经济流动、事件影响。支持大地图缩放、拖动、点击城市进入管理界面。

### 技术实现路径
- **Web 原型** : HTML5 Canvas + Konva.js 或 SVG + GSAP 动画。城市作为可点击 Node，部队使用小图标动画移动。
- **完整游戏** : Godot 4 TileMap + Node2D 或 Unity 地图系统。支持层（地形、城市、部队、效果、Fog of War）。
- **实时更新机制** : 每帧/回合结束时更新城市状态、部队位置、资源流动。使用动画表现移动、围攻、贸易车队。

### 地图要素
- **城市节点** (10-20 个核心) : 临安、建康、襄阳、鄂州、明州等。每个城市显示：城墙图标、人口数字、流程程度条、忧慮度颜色、在场部队数量、是否被围攻。
- **连接线** : 长江、钱塘江、道路、山脉阻碍。水路支持船队动画。
- **动态层** :
  - 部队移动箭头与小图标动画
  - 贸易车队/船队脉动线
  - 政策效果覆盖色（稳定度绿、战争度红、经济流通蓝）
  - 事件弹出标记和动画
  - Fog of War 边缘模糊
- **小地图** : 可缩放大地图的缩略版，显示全局势力分布。

### 与其他系统集成
- **官员与地图** : 将领带队在地图上移动，点击城市分配官员或解围。
- **政策树** : 解锁后直接在地图上显示新建筑物、改造动画、经济流动增强线。
- **战斗** : 地图上点击可进入战场或自动模拟结果。围攻时城市图标出现火光、损坏动画。
- **多国家** : 地图上显示金国、蒙元、大理等势力区域和军队移动箭头。

## 二、 多参考国家/势力系统扩展

** 核心势力 ** :
- **南宋** (玩家) : 绿色系，多城市管理、水网防御、内政拉拽。
- **金国** (主要敌人，前中期) : 黄红系。内部完颜氏派系拉拽、军事重建、经济抢劫。可触发内部事件影响入侵强度。
- **蒙元/元朝** (后期主要威胁) : 暗红/褐色系。巨大部队滚雪球机械、长期围围襄阳等战略、海上封锁。随时间推进战力暴增。
- **大理国** (西南参考) : 青绿系。可外交、贸易、联合抗蒙或被金/蒙利用。
- **其他参考势力** : 海上贸易伙伴、假齐国、内部主战/主和派隐形影响、西夏残余势力。

### 多国家机械
- **AI 行为** : 各国家有独立政策树、经济模拟、军事行动。金国会内斗影响入侵强度。蒙元有“滚雪球”机械。
- **外交与联合** : 可派官员外交、签订盟约、联合军事行动。
- **多线战场** : 玩家需要同时应对金国侵略、蒙元压力、内部稳定。
