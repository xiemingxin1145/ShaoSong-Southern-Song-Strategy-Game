# 《绍宋：南渡无悔》 游戏设计文档 (GDD) v1.0

**项目概述** : 一款深度南宋主题策略游戏，融合《无悔入华夏》的经济政策系统、《三国志》的官员地图与战斗机械、《绍宋》小说的架空历史与反爽文厚重感。

## 1. 核心玩法与目标
- **玩法核心** : 回合制大地图经营、官员招募与任命、城市建设与经济、朝堂政治斗争、战略/战术战斗。
- **胜利条件** : 统一江南、防守或北伐成功、经济文化繁荣、或在架空模式中推进改革实现《绍宋》风格的科学与军事现代化。
- **历史基础** : 从1127年建炎南渡开始，关键节点包括黄天沛、采石之战、邾城之战、绍兴和议、后期蒙元入侵、崖山海战。

## 2. 地图与城市系统
** 地图类型 ** : 城市/府州节点连接图（河流、山脉、沿海交通线），可扩展为六角或省份分区。

**核心城市列表** (搜索整理，可扩展至数十个关键节点):
- **临安府** (杭州): 首都，经济文化中心，人口巨大（南宋时世界大都市），西湖、宫城、商业繁荣。特殊建筑：帝宫、太学、水运。
- **建康府** (南京): 留都，战略前线，长江防线。
- **襄阳府**: 北防重镇，后期蒙元主攻目标。
- **鄂州 / 武昌**: 湖北重镇，江湖交通。
- **明州 / 海口城市**: 海外贸易、海军基地。
- 其他: 绍兴府、平江府、吉州等。

** 城市机械 ** :
- 人口、税收、农业/丝业/盐业/海贸生产。
- 建筑: 城墙、营房、学宫、码头、港口、防御工事。
- 特殊: 临安文化值高、海军系统。

## 3. 武将文臣系统 (几千名规划)
** 数据库结构 ** : JSON 或 SQLite。字段: id, name, type (武/文), stats (武力, 统率, 智力, 政治, 魅力, 忠诚, 野心), skills (e.g. "抗金", "水战", "科举", "内政"), background, portrait_path, faction_tendency (主战/主和/中立), historical_accuracy.

** 核心历史官员样例 ** (搜索整理，可扩展至数千):
** 中兴四将及其他武将 **:
- 岳飞: 高武力/统率，忠义高，技能「岳家军」士气加成、步兵强大。
- 韩世忠: 高武力，水战专家（黄天沛大胜），胆大。
- 张俊 / 刘光世 / 刘锑 / 吴玠 / 吴璞: 各有特长。

** 文臣 **:
- 秦柧: 高政治，主和派领袖，内政/和议系统加成，但忠诚低或引起不稳定。
- 尤允文: 文臣但具军事才能（采石之战），高智力。
- 李纲、赵鼎、文天祥、陆秀夫: 各具特色。
- 贿似道: 晚期权相，军事能力强但腐败风险高。

** 扩展至几千名 **: 
- 历史记录中的官员、将领、科举士子、地方武装。
- 生成系统: 基于朝代、地域、阶层生成名字、统计倾向。
- 游戏中可招募、会试、事件触发新官员。

## 4. 政治与派系
- 主战派 vs 主和派（秦柧等）：影响和议、军费、稳定度、北伐决策。
- 皇帝态度、朝堂辩论、限制大将兵权（重文轻武历史特色）。
- 政策树: 经济、军事、外交、科举改革、海军建设等。

## 5. 战斗系统
- 战略地图战斗: 将领带队移动、占领城市、补给。
- 战术战斗: 简化或全面模拟（步兵、弩兵、水军重要），将领技能、士气、地形影响。
- 海战与河战特别机械。

## 6. 艺术方向与插图规划 (几百张)
** 风格 ** : 精美中国风格，深度南宋美学（水墨画风 + 彩色细节）或现实主义游戏美术。
** 批量生成管道 ** : 
- 武将像: 铀甲、帽子、表情、武器、背景微微战场。
- 文臣像: 官服、头巾、手持朝策或笔墨。
- 城市景观: 临安详细街市、西湖、宫城；建康防御、江湖风光。
- 事件CG: 黄天沛水战、岳飞北伐、朝堂辩论、崖山海战。
- 单位/卡片: 军队图标、建筑。

** 已生成示例 ** : 见 art/ 或 repo images。
